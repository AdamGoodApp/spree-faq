An Spree Commerce extension for managing FAQs.

## Installation

Add to your `Gemfile`:

```ruby
gem 'spree-faq', github: 'futhr/spree-faq', require: 'spree_faq', branch: 'master'
```

Then run from the command line:

```sh
bundle && bundle exec rails g spree_faq:install
```
